# 净空间与以太物理模型

本词条记录C17星区世界观下，关于“净空间 (Net Space)”及其内部填充介质“以太 (Ether)”的核心物理法则与航行工程学基础。

## 基础物理属性

**以太的物质形态**
以太并非虚无的空间背景。它是净空间内广泛存在的一种物理实体，形态类似太空尘埃。以太具有明确的流体力学特征，对高速运动的物体会产生类似空气阻力的作用。整体密度偏低，常规速度下阻力可忽略，仅在极高速度下产生显著影响。

**速度常数与激波**
净空间内的光速是绝对的。电磁波（光与通讯信号）在净空间内遵循普通物理规律传播，但其传播速度随净空间极高的光速上限而同步提升。
由于以太具备介质属性，物体极速运动时会压缩以太产生激波。净空间存在“以太音障”，临界点约为现实光速的 **8752.6倍**。

**以太气象**
以太流体会形成类似行星大气系统的“以太风”。其成因目前未知，系统极度混沌，无法进行长期精准预测。深空航行中存在气象剧烈的危险区，会对飞船进气和航线造成严重干扰。

## 空间架构与叠加态

**下潜深度 (Diving Depth)**
飞船进入净空间的状态用“深度”标定，区间为 0 到 1。受限于当前技术，绝对深度 1 无法到达。
* **浅层 (接近0)**：以太密度低。受现实宇宙大质量天体（如恒星）的引力投影影响极大。
* **深层 (接近1)**：以太密度极高，阻力剧增。但高密度以太意味着引擎能吸入更多工质，推力上限成倍提升。现实天体影响降至极低。

**双空间叠加态**
飞船必须依靠系统持续供能，主动维持“叠加态”才能滞留净空间。
如果系统供能中断或发生故障，叠加态会瞬间崩溃。飞船将被强制弹回现实空间（瞬时减压脱出）。若在极深的高压区发生强制弹回，引擎本体可能因无法承受压力骤变而严重损毁。

## 动力学与工程学

**吸入式以太喷气推进**
本宇宙不存在曲率航行。净空间航行原理等同于大气层内的喷气式发动机。
引擎前端吸入净空间中的以太尘埃，经过内部加速后向后喷射，依靠反作用力推进。进气量直接决定飞船推力。遭遇以太乱流或大角度机动导致进气不足时，飞船会立刻面临推力骤降和偏航风险。

**能源供给**
“特制氢”是核聚变反应堆的唯一燃料，仅负责提供庞大的热能与电能。特制氢不参与喷射，不是推进工质。

**气动布局设计**
CTX星际科技集团的飞船普遍采用极简白色几何体与锐利边缘设计。这是为了适应高亚音速与突破以太音障时的极端气流环境。飞船外壳大量采用类似超音速战机的后掠翼结构，用于切割以太流体、减小激波阻力。

## 导航与侦查战术

**航迹扰动与声学侦查**
* **前置效应**：飞船推进时会挤压前方以太，产生极具特征的压力波。
* **尾迹**：任何移动都会留下以太波浪尾迹。
* **超音速暴露**：速度一旦突破 8752.6c 的以太音障，扰动波浪呈指数级增强，极易被远距离侦测。

**隐身与姿态控制**
由于以太平时密度极低，利用纯气动舵面控制姿态非常迟钝。飞船主要依靠引擎喷口的矢量推力转向，气动外形仅用于高速下的姿态稳定。现代理论隐身学主要集中于雷达探测算法的逆向工程与规避计算。

**信标网络与盲飞**
导航信标是锚定在净空间的常驻电磁波发射灯塔。依赖净空间的高光速，实现极低延迟的信号广播。
当飞船脱离信标覆盖区（或信标被摧毁）进入“盲飞”状态时，唯一可靠的定位手段是主动上浮切回现实空间，通过光学观测实宇宙星图重新校准坐标。
